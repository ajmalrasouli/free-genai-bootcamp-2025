Project Guide

Quick overview of our learning tools

1️⃣ Farsi Text Adventure MUD Game

Immersive text-based adventure game for practicing Farsi in an interactive virtual world.

2️⃣ ASL Finger Spelling Application

Interactive tool for learning and practicing American Sign Language finger spelling.

3️⃣ Language Learning Portal

Centralized platform for accessing all language learning resources and tracking progress.

4️⃣ Persian Learning Assistant

Interactive listening comprehension system for Persian language learning.

5️⃣ Text-to-Speech Microservice

Microservice for converting text to speech in multiple languages.

6️⃣ Farsi Song Vocabulary Generator

Generate vocabulary lists from Farsi songs for language learning.

7️⃣ Farsi Learning Visual Novel

Interactive visual novel for immersive Farsi language learning.

8️⃣ Farsi Writing Practice App

Interactive writing practice application for Farsi language learners.
